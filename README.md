# GObjob

## Setup

### Creating the config
Inside `<MAIN>` create a copy of `config.example.hcl`

Then change the variables to whats needed

## Run

Inside `<MAIN>` run `go run main.go`

### Run in prod

To run for prod, inside `<MAIN>` run `sudo GIN_MODE=release go run main.go`
